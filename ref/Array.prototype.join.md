== Array.prototype.join


`join()` turns an array into a string. The `separator` parameter is a string with comma as the default value.

```
[1, 2, 3].join(); //:: "1,2,3"
[1, 2, 3].join('|'); //:: "1|2|3"
[1, 2, 3].join(' is less than '); //:: "1 is less than 2 is less than 3"
```
